## ğŸ“¦ Installation

1.  **Clone the repository**
    ```bash
    git clone <repository-url>
    cd <project-folder>
    ```

2.  **Install Dependencies**
    *   **Backend:**
        ```bash
        npm install
        ```
    *   **Frontend:**
        ```bash

        npm install
        ```

4.  **Run the Application**
    *   **Backend:** `npm start` (runs on port 3000)
    *   **Frontend:** `npm run dev` (runs on port 5173)
