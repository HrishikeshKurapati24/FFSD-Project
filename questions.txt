Backend Interview Questions for FFSD Project

1. Authentication & JWT:
   - Where is JWT token verification implemented in the backend?
   - Explain the verifyJWTFromCookie function and how it's used in middleware.
   - How does the isAuthenticated middleware work and what does it check?
   - Where are JWT tokens generated and set as cookies?
   - What happens when a JWT token expires?

2. Cart Logic:
   - Where is the cart functionality implemented in the backend?
   - Explain how the session-based cart works in customerRoutes.js.
   - How does the addToCart function validate product availability?
   - Where is cart data stored and how is it managed across requests?
   - Explain the checkout process and how it handles stock management.

3. Middleware & Security:
   - What middleware is used for authentication across different user types (brand, influencer, customer)?
   - How is CORS configured in the application?
   - What security measures are implemented (helmet, session config, etc.)?
   - How does the application handle both session and JWT authentication?

4. Database & Models:
   - How are the MongoDB models structured (Brand, Influencer, Customer, etc.)?
   - Where is database connection and initialization handled?
   - How are user passwords hashed and verified?
   - Explain the relationship between campaigns, products, and influencers.

5. API Routes & Controllers:
   - How are routes organized for different user types?
   - Explain the customer purchase controller and its main functions.
   - How does the application handle file uploads and cloud storage?
   - Where is notification handling implemented?

6. Session Management:
   - How are user sessions configured and managed?
   - What data is stored in sessions vs JWT tokens?
   - How does the application handle session cleanup?

7. Error Handling & Validation:
   - How are errors handled across the application?
   - What validation is performed on user inputs?
   - How does the application handle API vs page requests?

8. Business Logic:
   - How are campaigns activated and managed?
   - Explain the product stock management system.
   - How are rankings calculated for brands and influencers?
   - What is the subscription system and how does it work?

9. Third-party Integrations:
   - How is Cloudinary integrated for image uploads?
   - What notification services are used and how?
   - How are social media platforms handled for influencers?

10. Performance & Scalability:
    - How are database queries optimized?
    - What caching mechanisms are implemented?
    - How does the application handle concurrent requests?
    - What measures are taken for production deployment?
